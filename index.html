sinc(x) / x Visualization

This repository generates a plot of the function sin(x) / x over the interval -4π to 4π. The image shows the curve, grid lines, axis labels, and shaded positive and negative regions.

The expression is usually written as sinc(x). At x = 0, the code uses the limit value 1 so the function stays well-defined.

How the code works

  • It creates a dense sample of x values with PyTorch.
  • It computes y = sin(x) / x, using 1 at x = 0.
  • It maps the data to pixel coordinates on a fixed canvas.
  • It draws a dark background, grid lines, axes, labels, and the curve.
  • It shades positive values green and negative values red.
  • It saves the final figure as sinc.jpg.

Why this function matters

The sinc function appears in signal processing, Fourier analysis, and many interpolation problems. It is a simple example of a function with oscillating decay and a removable singularity at zero.

The plot highlights the central peak at x = 0, where the limit is 1, and the alternating side lobes that decay as |x| grows.

Implementation notes

The plot is generated with torch for sampling and Pillow for image rendering. The final output is a static image, not an interactive chart.

Files in this repo
